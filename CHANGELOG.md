# Changelog

## 2.4.0 - 2026-05-26

### Added

- 新增后台 AJAX 队列清理。
- 支持选择多个清理项目自动分批执行。
- 新增清理进度条、实时日志、暂停/继续按钮。
- 队列任务写入维护日志，并自动清除诊断报告缓存。

## 2.3.0 - 2026-05-26

### Added

- 新增 GitHub Release 插件内自动更新支持。
- 支持 WordPress 后台检测 GitHub 最新版本。
- 支持插件详情弹窗展示 GitHub Release 更新说明。
- 支持从 Release 附件 `wp-large-content-optimizer.zip` 下载更新包。

## 2.2.0 - 2026-05-26

### Changed

- 后台界面改为 Tab 分组展示。
- Tab 分组包括：概览、数据库、采集站、前台优化、日志、设置。
- 支持记住上次打开的 Tab。
- 不再把所有模块堆在一个长页面中。

## 2.1.0 - 2026-05-26

### Changed

- 优化后台 UI。
- 新增顶部 Hero 概览与健康评分展示。
- 新增快捷导航。
- 支持模块折叠/展开，默认折叠长模块。
- 优化卡片、表格、按钮、风险标签和移动端布局。

## 2.0.0 - 2026-05-25

### Added

- JSON 诊断报告导出。
- 导出内容包含插件版本、导出时间、站点 URL、完整诊断报告、当前设置和维护日志。
- 导出操作写入维护日志。

## 1.9.0 - 2026-05-25

### Added

- 数据库维护日志。
- 记录最近 100 条维护操作。
- 后台展示操作时间、管理员、action 和结果消息。
- 支持清空维护日志。

## 1.8.0 - 2026-05-25

### Added

- 数据库慢查询风险分析。
- post_type 分布。
- post_status 分布。
- taxonomy 关系 TOP。
- autoload 总体积。
- 低选择性 meta_key 风险。
- 自动生成慢查询风险建议。

## 1.7.0 - 2026-05-25

### Added

- 前台性能与缓存检测。
- 对象缓存、WP_CACHE、WP-Cron、自动保存间隔、WP_POST_REVISIONS 状态检测。
- 前台轻量优化设置：禁用 Emoji、可选禁用 oEmbed、访客禁用 Dashicons、移除 generator 标签。

## 1.6.0 - 2026-05-25

### Added

- 后台大列表快速模式。
- 限制后台文章/页面列表每页数量。
- 关闭后台列表 meta/term cache 预加载。
- 后台搜索优先按标题搜索。
- 禁用月份下拉统计。
- 可选禁用精确总数统计。

## 1.5.0 - 2026-05-25

### Added

- 已发布重复文章审查器。
- 展示重复标题组中每篇文章 ID、发布时间、字数、缩略图状态、查看链接、编辑链接。
- 只展示，不自动处理已发布文章。

## 1.4.0 - 2026-05-25

### Added

- 重复文章处理工具。
- 重复草稿分组报告。
- 每组保留最早一篇，其余重复草稿移入回收站。
- 仅处理 draft、auto-draft、pending，不处理 publish。

## 1.3.0 - 2026-05-25

### Added

- 采集站专项体检。
- 重复标题组、重复 GUID 组、空内容文章、短内容文章、已发布无缩略图、采集失败草稿统计。
- 短内容阈值设置。
- 采集失败草稿分批清理。

## 1.2.0 - 2026-05-25

### Added

- 安全优化向导。
- 缓存与环境检查。
- 按低/中/高风险展示推荐执行顺序。

## 1.1.0 - 2026-05-25

### Added

- 性能诊断评分。
- 优化建议。
- 数据表大小 TOP。
- postmeta 热点字段 TOP。
- autoload 大对象 TOP。
- 诊断报告 10 分钟缓存与手动刷新。

## 1.0.0 - 2026-05-25

### Added

- 初始版本。
- 数据库体检。
- 分批清理 revision、auto-draft、trash、孤儿 postmeta、孤儿分类关系、过期 transient。
- 推荐索引检测与添加。
- 后台文章/页面列表轻量化。
- 修订版本限制。
- 每日自动维护设置。
